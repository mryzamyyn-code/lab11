React SPA: Список задач 

## Цель работы
Создать простое одностраничное приложение (SPA) на React, которое позволяет:
- отображать список задач;
- добавлять новые задачи через форму;
- удалять задачи из списка;
- (опционально) сохранять задачи в LocalStorage, чтобы они сохранялись при перезагрузке страницы.

## Описание приложения
Приложение состоит из трёх основных компонентов:
1. **App.jsx** — главный компонент, который объединяет форму и список задач.
2. **TaskForm.jsx** — компонент формы для добавления новой задачи.
3. **TaskList.jsx** — компонент для отображения списка задач с возможностью удаления.

Приложение использует React для управления состоянием и Vite для сборки проекта.
